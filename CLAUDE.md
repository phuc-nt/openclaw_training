# OpenClaw Training — Agent Instructions

Training materials for onboarding teams (marketing/sales) to OpenClaw and Claude Desktop.

When opened from `openclaw-workspace/`, the parent CLAUDE.md defines admin identity and universal rules.

## Content Guidelines

- Two deployment tracks: **OpenClaw (self-hosted)** and **Claude Desktop (Cowork)**
- Core principle: "80% of agent quality depends on SOUL.md writing, not code"
- Slides use Reveal.js — edit `slides.html`, view in browser
- All examples must use generic placeholders, never real credentials or personal data

## Structure

```
├── 01-overview/                  ← AI agent intro, solution comparison, pre-training survey
├── 02-openclaw-training/         ← 5 training modules + roles (OpenClaw self-hosted)
├── 03-use-cases/                 ← Practical scenarios (immediate + roadmap)
└── slides.html                   ← Reveal.js presentation
```
